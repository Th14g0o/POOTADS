# Ler

## stream: 
A partir da criação do objeto Scanner com System.in como parâmetro ele  já começa a ler. Esse objeto vem do java.util.Scanner.

Para pegar esses valores se usa uma função do objeto NextInt por exemplo pega o próximo valor e tenta usar como inteiro se não conseguir erro

## Importar
```java
import java.util.Scanner;
```

## Declaração
```java
Scanner sc = new Scanner(System.in);
```

## Metodos:
* Scanner var.nextLine(); pega linha inteira no terminal
* Scanner var.next(); pega proxima string
* Scanner var.nextInt(); pega proximo inteiro
* Scanner var.nextDouble(); pegra procimo numero fracionario
